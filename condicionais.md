# CONDICIONAL
Condicional pode ser entendido como uma escolha.
Para fazer essas escolhas usamos operadores logicos ou matematicos.
Exemplos:
- +,-,*,/   
- <,>,==
- and(&&),or(||),not(!)

As palavras reservadas para isso sao: _if_, _else_ , _else if_
## IF
A condiçao _if_ significa se.
Se atender as condições faça uma coisa.
## ELSE 
A condição _else_ significa _senão_, quer dizer que a primeira condição não foi aceita e agora estamos no _else_.
## ELSE IF
A condição _senão se_.
Essa condição não pode ser utilizada nem como a primeira e nem ultima condição pois representa como se fosse uma condição intermediária.
# OPERADOR TERNÁRIO
Essa operação é bem parecida com _if_ _e_ _else_, contudo usaremos a interrogação que significa _if_ ? e else significa :
- O simbolo de porcentagem % significa modulo ou resto.
- Os dois iguais == significa compara uma coisa com a outra.
- o simbolo de === em javascript significa compara valor e tipo.
